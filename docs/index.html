Quick Start

1

Download & Install

Download ArchiveSetup.exe and run the installer. Or download the portable version - no installation required.

2

Create a Backup Job

Click "Add Job" and configure your source and destination folders, sync options, and exclusion patterns.

3

Set Your Schedule

Enable scheduling to run backups automatically on specific days and time windows, or run jobs manually anytime.
