Installation

  1. Download the installer (ArchiveSetup.exe) from a release page.
  2. Run the installer.
  3. Launch Archive from the Start menu.

Safety checks

Before installing:

  • Confirm the download URL starts with https://github.com/Ignyos/Archive/releases.
  • Confirm the file name matches the release asset listed on GitHub.

If your environment requires it, verify checksums/signing based on your organization policy.
