Troubleshooting

Common issues and fixes for setup, launch, and preview workflows.

Preview shows access denied

Example:

Unable to generate preview: Access to the path 'E:\System Volume Information' is denied.

Why this happens:

  • System Volume Information is a Windows-protected folder.
  • Access can be denied even when the drive appears writable.

What to do:

  • Use a specific source folder instead of the drive root.
  • Keep Skip hidden and system files enabled when possible.
  • Re-run preview.

App does not start after build

  • Use the VS Code launch profile Run Archive (Clean & Build).
  • Confirm build succeeds.
  • If needed, close any old running Archive.Desktop.exe process and retry.

Quartz scheduler serializer error

If you see a startup error about quartz.serializer.type or JSON serializer instantiation:

  • Update to latest release.
  • Ensure installation files come from official releases.
  • If building from source, restore and rebuild dependencies.

Reporting an issue

Open an issue with:

  1. Archive version
  2. Windows version
  3. Steps to reproduce
  4. Full error message
Include exact error text in issue reports to speed up diagnosis.
