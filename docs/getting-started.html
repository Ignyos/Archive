Getting Started

Install Archive, configure your first job, and validate results safely.

1) Install Archive

  • Download from: https://github.com/Ignyos/Archive/releases/latest
  • Run ArchiveSetup.exe

2) Create your first job

  1. Open Archive.
  2. Select New Job.
  3. Enter a job name.
  4. Pick a source path (file or folder).
  5. Pick a destination folder.
  6. Choose schedule type (Manual, One-time, or Recurring).
  7. Save.

3) Preview operations

Use Preview Operations before running a job.

  • Preview uses your configured sync options.
  • Protected Windows system folders may be skipped automatically.

4) Run and verify

  • Run the job manually from the app.
  • Confirm copied/updated files at destination.
  • Review execution history/logs in the app.

Tips

  • Avoid using a whole drive root (like E:\) as source when possible.
  • Prefer a specific top-level folder to reduce permission-related skips.
  • Keep destination outside source to avoid recursive copy behavior.
